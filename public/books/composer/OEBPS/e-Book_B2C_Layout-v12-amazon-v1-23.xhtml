Black notes

Time for the black notes. Did you know black notes have two names? When we move from C to the next key up, we call that black note C sharp (C# for short). When we move from D to the next key down, we call that same black note D flat (aka Db). Can you find all the C#’s? All the Db’s? (Gotcha, they’re the same keys). How about all the F#’s? The Gb’s? Now you’re getting it.

A rose by any other name

C# is the enharmonic equivalent of Db. An enharmonic equivalent is simply a note that shares the same pitch as another note, though it bears a different name. Big words for a deceptively simple concept.
